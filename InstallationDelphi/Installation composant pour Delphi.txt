------------------------------------------------
-- Installation de la RX Library dans Delphi 7 --
------------------------------------------------
1 - Installer la RX Library dans le répertoire voulu.
2 - Ouvrir Delphi.
3 - Fichier->Tout Fermer
4 - Composant->Installer des paquets
                        dans "paquet de conception" chercher ceux de la RX et les supprimer
5 - Fichier->Ouvrir (dans l'ordre)
                 Dans le répertoire \Units de l'installation de la RX
	   rxctl7.dpk  : Option =>"Option d'utilisation"->conception et execution  OK
                                        Compiler
                                        Installer (message de confirmation)
                     rxdb7.dpk  : Option =>"Option d'utilisation"->conception et execution OK
                                         Compiler
                                         Installer (message de confirmation)
                    rxdbe7.dpk : Option =>"Option d'utilisation"->conception et execution OK
		    Compiler
                                         Installer (message de confirmation)
                   -----------------------------------------------------------------------------
                   dclrx7.dpk     : Compiler
                                           Installer (message avec le nom des nouveaux composants)
                   dclrxdb7.dpk : Compiler
                                           Installer (message avec le nom des nouveaux composants)
                   dclrxbd7.dpk : Compiler
                                           Installer (message avec le nom des nouveaux composants)
6 - Fichier->Quitter (Ne pas enregistrer les modifications)
