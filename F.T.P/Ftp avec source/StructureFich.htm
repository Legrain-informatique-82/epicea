Structure d'un fichier paramètre

Les fichiers paramètres sont des fichiers texte (*.txt)

1 PARAMETRE:
Seules les 6 premières lignes sont prises en compte.
1ère ligne : Répertoire source
2ème ligne : Répertoire destination
3ème ligne : Utilisateur
4ème ligne : Mot de Passe
5ème ligne : Adresse du serveur FTP
6ème ligne : Si 'o' ou 'O' supprime le contenu du repertoire destination avant le transfert

Exemple:
C:\A_Envoyer
/sauvegarde
utilisateur
password
serveurFTP.fr
n

PLUS D'UN PARAMETRE:
Seules les 3 premières lignes sont prises en compte.
3ème ligne : Utilisateur
4ème ligne : Mot de Passe
5ème ligne : Adresse du serveur FTP

Exemple:
utilisateur
password
serveurFTP.fr
