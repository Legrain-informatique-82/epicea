Modes d'utilisation

AUCUN PARAMETRE:
Lance le programme avec l'interface graphique complète.
Avec au moins un paramètre seul certains bouton et les barres de progression sont présent.

1 PARAMETRE:
Le paramètre doit être 'a' ou 'A' et le fichier FTPparam.txt doit contenir
tous les paramètres de connexions et de transfert.
Ex: envoi_ftp a

4 PARAMETRES:
Source + destination + Fichier texte contenant les paramètres de connexion + Effacer
Ex: envoi_ftp Repertoire_Source Repertoire_Destination Param.txt o
      envoi_ftp Repertoire_Source Repertoire_Destination Param.txt n

5 PARAMETRES:
Source + destination + Fichier texte contenant les paramètres de connexion + Effacer + Auto
Ex: envoi_ftp Repertoire_Source Repertoire_Destination Param.txt o a
      envoi_ftp Repertoire_Source Repertoire_Destination Param.txt n a

6 PARAMETRES:
Source + Destination + Login + PassWord + Adresse + Effacer
Ex: envoi_ftp Repertoire_Source Repertoire_Destination Login Mot_De_Passe Adresse_Du_Serveur_FTP o
      envoi_ftp Repertoire_Source Repertoire_Destination Login Mot_De_Passe Adresse_Du_Serveur_FTP n

7 PARAMETRES:
Source + Destination + Login + PassWord + Adresse + Effacer + Auto
Ex: envoi_ftp Repertoire_Source Repertoire_Destination Login Mot_De_Passe Adresse_Du_Serveur_FTP o a
      envoi_ftp Repertoire_Source Repertoire_Destination Login Mot_De_Passe Adresse_Du_Serveur_FTP n a
